## 特性

### 比较完整

预计有超过 40 个组件, 正在持续发开中。

支持 treeshaking。

### 使用 TypeScript

Amu UI 全量使用 TypeScript 编写，和你的 TypeScript 项目无缝衔接。

你不需要导入任何 CSS 就能让组件正常工作。


## 安装

### npm

使用 npm 安装。

```bash
npm i -D amu-ui
```
